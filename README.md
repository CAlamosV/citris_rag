# Flask App Setup

## Create and Activate Virtual Environment

1. Create a virtual environment:

   ```sh
   python3 -m venv venv
   ```

2. Activate the virtual environment:
   - On macOS/Linux:
     ```sh
     source venv/bin/activate
     ```
   - On Windows:
     ```sh
     .\venv\Scripts\activate
     ```
